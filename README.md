## 🌟 專案核心價值

本專案不僅是一個靜態網頁，而是一個具備內容管理功能的服務平台，旨在解決以下需求：
1. **靈異紀實與傳承**：將親身經歷與所見之靈界故事數位化，提供沉浸式閱讀體驗。
2. **專業靈學諮詢**：提供透明、專業的服務項目資訊，為有需要的讀者提供協助管道。
3. **動態內容管理**：透過後端自動化技術，即時更新「最新公告」與「服務項目」。

---

## 🛠️ 網站技術特色 (SEO 優化)

本站採用現代前端與自動化技術架構，確保搜尋引擎友善度 (SEO Friendly)：

*   **語義化 HTML5**：嚴格遵循 HTML5 結構，幫助 Google 爬蟲理解服務重點與公告內容。
*   **社群 Meta 標籤優化**：完整配置 `OG (Open Graph)` 與 `Twitter Card`，確保在 Facebook 與 LINE 分享時擁有美觀的預覽圖片與精確描述。
*   **動態資料驅動**：
    *   使用 `ES6 JavaScript (Modules)` 非同步獲取最新的公告與服務清單。
    *   結合 Google Apps Script (GAS) 達成資料庫自動同步，無需頻繁手動改稿。
*   **響應式設計 (RWD)**：支援手機與平板流暢閱讀，符合 Google 「行動裝置優先」的排名準則。
